# 🌌 ORIS OS - Neon Glass UI

**ORIS OS** é uma interface web inspirada no estilo futurista e minimalista do Spotify, com tema escuro (Dark Mode), design em vidro fosco (Neon Glass) e navegação lateral intuitiva. Ideal para dashboards, painéis administrativos ou sistemas operacionais baseados na web.

---

## 📂 Páginas Inclusas

| Página         | Descrição                                         |
|----------------|--------------------------------------------------|
| `dashboard.html`     | Página principal com visão geral do sistema     |
| `notificacoes.html`  | Exibe notificações e alertas                    |
| `perfil.html`        | Edição de dados pessoais do usuário             |
| `configuracoes.html` | Preferências e configurações do sistema         |
| `sobre.html`         | Informações institucionais e créditos           |
| `login.html`         | Tela de autenticação com design moderno         |

> 🔸 A página **Meta Rabbit** não está inclusa neste projeto.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**  
- **CSS3** (com estilo embutido)
- **JavaScript** (para interações básicas)
- **Google Material Icons** (ícones modernos e leves)

## 🚀 Como Utilizar

1. **Clone** este repositório ou faça o download do `.zip`.
2. Abra qualquer arquivo `.html` no seu navegador.
3. Navegue pelas páginas através da **sidebar lateral**.
4. Clique em **Logout** para simular a saída do sistema.

---

## 🎨 Personalização

Você pode personalizar facilmente:

- Cores e temas (alterando o CSS embutido nos arquivos HTML)
- Ícones (via [Google Material Icons](https://fonts.google.com/icons))
- Layout e organização de cada página
- Scripts de autenticação real (caso deseje integrar com Firebase, GitHub OAuth, etc.)
