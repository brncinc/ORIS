# ORIS OS - Neon Glass UI

ORIS OS é uma interface web inspirada no design moderno do Spotify, com tema escuro e navegação lateral (sidebar). O projeto contém várias páginas essenciais para um sistema operacional web ou dashboard, com foco em usabilidade, acessibilidade e design elegante.

---

## Páginas incluídas

- **Dashboard**: Página principal com resumo das informações.
- **Notificações**: Lista de alertas e notificações do sistema.
- **Perfil**: Visualização e edição de dados pessoais do usuário.
- **Configurações**: Ajustes e preferências do sistema.
- **Sobre**: Informações sobre o ORIS OS.
- **Login**: Página para autenticação do usuário.

> A página **Meta Rabbit** não está incluída neste projeto.

---

## Tecnologias usadas

- HTML5
- CSS3 (estilos inline nos arquivos)
- Google Material Icons para os ícones da navbar
- JavaScript básico para navegação e logout simulado

---

---

## Como usar

1. Clone ou baixe o repositório.
2. Abra os arquivos `.html` em seu navegador favorito.
3. Navegue pelo menu lateral para acessar as páginas.
4. Use o botão **Logout** para simular saída e voltar para a página de login.

---

## Personalização

Você pode adaptar o layout, cores e conteúdos conforme suas necessidades. O CSS está embutido nos arquivos para facilitar a portabilidade.
